まずは、いちばん簡単なプログラムを書いてみましょう。

以下のようなプログラムを実行すると、日本の国旗のような画像が表示されます。

```javascript
function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(255, 255, 255);
  fill(255, 0, 0);
  ellipse(200, 200, 200, 200);
}
```

完成したら、実行ボタンをクリックして、実行結果をみてみましょう。
