## 学習内容

- `createCanvas()` - キャンバスの作成
- `background()` - 背景色の指定（RGB: 0-255）
- `fill()` - 塗りつぶし色の指定
- `ellipse()` - 円の描画（x, y, width, height）
- コメント `//` の使い方

## 説明

```javascript
function setup() {
  createCanvas(400, 400); // 400x400 のキャンバスを作成
}

function draw() {
  background(255, 255, 255); // 白い背景
  fill(255, 0, 0); // 赤色で塗りつぶし
  ellipse(200, 200, 200, 200); // 中央に円を描画
}
```

### 座標系

左上が (0, 0)、右下が (400, 400)

### RGB 色の表現

加法混合で色を表現する:

- `255, 255, 255` → 白
- `0, 0, 0` → 黒
- `255, 0, 0` → 赤
