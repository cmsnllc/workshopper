## 複数の図形を描く

前回は1つの円を描いたけど、今度は複数の図形を描いてみよう。

### 色の変更タイミング

`fill()` を呼ぶと、**それ以降に描く図形がすべてその色になる**。

```javascript
fill(255, 0, 0);  // ここから赤色
ellipse(100, 200, 50, 50);  // 赤い円

fill(0, 0, 255);  // ここから青色
ellipse(200, 200, 50, 50);  // 青い円
ellipse(300, 200, 50, 50);  // これも青い円
```

### 重なりの順序

プログラムは**上から順番に実行される**ので、後に描いた図形が前に描いた図形の上に重なる。

```javascript
// 最初に大きい円を描く
fill(255, 0, 0);
ellipse(200, 200, 150, 150);

// 次に小さい円を描く（上に重なる）
fill(255, 255, 0);
ellipse(200, 200, 80, 80);
```

## 試してみよう

右のエディタに1つ目の円（赤）は既に描かれている。2つ目と3つ目の円を好きな色・位置で描いてみよう。
