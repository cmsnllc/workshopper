## 「もし○○のとき」

特定の条件の時のみ命令を実行したい時に使うのが if（もし○○のとき） です。

### if 文の基本

```javascript
if (条件) {
  // 条件が true のとき実行される
}
```

条件が `true` のときだけ、波括弧 `{}` の中のコードが実行されます。

### else を使う

```javascript
if (条件) {
  // 条件が true のとき実行される
} else {
  // 条件が false のとき実行される
}
```

`else` を使うと、条件が `false` のときの処理も書けます。

### 例：偶数・奇数の判定

```javascript
let x = 10;

if (x % 2 == 0) {
  print(x, "は偶数です");
} else {
  print(x, "は奇数です");
}
```

x が偶数なら「偶数です」、奇数なら「奇数です」と表示されます。

### 波括弧 {} の役割

波括弧 `{}` で囲まれた部分を**ブロック**といいます。ブロックの中には、複数の命令を書けます。

```javascript
if (x > 0) {
  print("x は正の数です");
  print("x の値は", x);
  print("計算結果:", x * 2);
}
```
