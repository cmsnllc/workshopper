## 👊チャレンジ⑬

**顔を描く関数 drawFace を完成させよう**

### 条件

- 顔は既に描かれている
- 左右の目を描く
- 口を描く
- どんな大きさの顔でも、バランスよく描かれるようにする

### ヒント

```javascript
function drawFace(x, y, size) {
  // 顔
  fill(255, 200, 100);
  ellipse(x, y, size, size);

  // 目（黒）
  fill(0);
  ellipse(x - size * 0.2, y - size * 0.1, size * 0.15, size * 0.15); // 左目
  ellipse(x + size * 0.2, y - size * 0.1, size * 0.15, size * 0.15); // 右目

  // 口
  ellipse(x, y + size * 0.2, size * 0.3, size * 0.1);
}
```

### ポイント

- **size を使って計算**すると、どの大きさでもバランスが保たれる
  - `size * 0.2` は、顔の大きさの 20%
  - 小さい顔なら目も小さく、大きい顔なら目も大きくなる
