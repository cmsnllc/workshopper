## 👊チャレンジ⑤

**変数 x が偶数かどうか判定しよう**

### 偶数とは

- 2, 4, 6, 8, 10... のように、2 で割り切れる数
- 2 で割った**余りが 0** になる

### ヒント

余り演算子 `%` と比較演算子 `==` を使って判定できるよ。

```javascript
x % 2 == 0  // x を 2 で割った余りが 0 と等しいか？
```

これが `true` なら偶数、`false` なら奇数だよ。

### 試してみよう

変数 x の値を変えて、いろいろな数で試してみよう：
- `x = 10` → 偶数（true）
- `x = 7` → 奇数（false）
- `x = 100` → 偶数（true）
- `x = 1` → 奇数（false）
