## 自由制作

ここまでできたら、あとは自由にプログラムを書いてみよう！

### これまで学んだこと

- **図形**: ellipse, rect, triangle, line
- **色**: fill, stroke, strokeWeight
- **変数**: let, 計算
- **条件分岐**: if, else, &&, ||
- **ループ**: for, while
- **アニメーション**: setup, draw
- **ランダム**: random()
- **マウス**: mouseClicked, mouseX, mouseY
- **関数**: function
- **再帰**: 再帰関数、フラクタル

これらを組み合わせて、オリジナルの作品を作ってみよう。

### 作品例

1. **シェルピンスキのギャスケット** - 三角形のフラクタル
2. **動く波のパターン** - sin/cos を使った波
3. **お絵描きツール** - マウスの軌跡を描く
4. **弾むボール** - 物理シミュレーション
5. **迷路生成** - ランダムな迷路を作る
6. **パーティクルシステム** - たくさんの粒子を動かす
7. **万華鏡** - 対称な模様を描く
8. **ゲーム** - クリックして敵を倒すゲーム

### 使えそうな関数

```javascript
// 図形
triangle(x1, y1, x2, y2, x3, y3);  // 三角形
rect(x, y, width, height);         // 四角形
line(x1, y1, x2, y2);              // 線

// 三角関数
sin(angle);  // サイン
cos(angle);  // コサイン

// 線の設定
stroke(r, g, b);      // 線の色
strokeWeight(width);  // 線の太さ
noStroke();           // 線なし
noFill();             // 塗りつぶしなし

// その他
dist(x1, y1, x2, y2);  // 2点間の距離
```

### ヒント

- まずは簡単なものから始めよう
- うまくいかなかったら、小さく分割して試そう
- 他の人の作品を参考にするのもいい
- **失敗を恐れず、いろいろ試してみよう！**
