## 関数を作ろう

これまで使ってきた setup や draw は関数といいます。ellipse なども、関数です。

かっこの中で指定している x, y, dx, dy を引数といいます。また、これらはプリミティブ関数といってあらかじめ定義してくれている関数です。

それらとは別に、自分で関数を定義できます。

```javascript
function add(x, y) {
  return x + y;
}
```

自作関数の例

### 自作関数の呼び出し方

```javascript
function draw() {
  background(255, 255, 255);
  fill(255, 0, 0);
  ellipse(add(50, 100), add(50, 100), 50, 50);
}
```

プリミティブ関数と同様にかっこで呼び出せます。
上記では、 ellipse に対して 150 と 150 が渡されます。
