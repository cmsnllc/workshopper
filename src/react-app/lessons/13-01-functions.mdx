## 関数を作ろう

**関数**を作ると、同じ処理を何度も使い回せる。

### 関数の定義

```javascript
function drawCircle(x, y, size) {
  fill(255, 0, 0);
  ellipse(x, y, size, size);
}
```

- `function` で関数を定義
- `drawCircle` は関数名（自由に決められる）
- `(x, y, size)` は**引数**（パラメータ）
  - 関数に渡す値

### 関数の呼び出し

```javascript
drawCircle(100, 100, 80);
drawCircle(250, 250, 120);
```

- 関数名を書いて、括弧の中に引数を渡す
- 1回目: x=100, y=100, size=80 で円を描く
- 2回目: x=250, y=250, size=120 で円を描く

### 関数を使う利点

1. **コードが短くなる**: 同じ処理を何度も書かなくていい
2. **わかりやすい**: `drawCircle` という名前で、何をする処理かすぐわかる
3. **修正が楽**: 関数の中身を変えれば、全部の呼び出しに反映される

### 例：複雑な図形を描く

```javascript
function drawFace(x, y, size) {
  // 顔
  fill(255, 200, 100);
  ellipse(x, y, size, size);

  // 目
  fill(0);
  ellipse(x - size * 0.2, y - size * 0.1, size * 0.1, size * 0.1);
  ellipse(x + size * 0.2, y - size * 0.1, size * 0.1, size * 0.1);

  // 口
  ellipse(x, y + size * 0.2, size * 0.3, size * 0.1);
}

// 使い方
drawFace(100, 100, 80);  // 小さい顔
drawFace(300, 300, 150); // 大きい顔
```

複雑な図形も、関数にすれば簡単に再利用できる。
