## 👊チャレンジ⑥

**x が 2 なら小さい赤い円、それ以外なら大きい青い円を描こう**

### 条件

- **x が 2 のとき**
  - 赤い円: `fill(255, 0, 0)`
  - 小さいサイズ: `ellipse(200, 200, 100, 100)`

- **それ以外のとき**
  - 青い円: `fill(0, 0, 255)`
  - 大きいサイズ: `ellipse(200, 200, 200, 200)`

### ヒント

```javascript
if (x == 2) {
  fill(255, 0, 0);           // 赤
  ellipse(200, 200, 100, 100); // 小さい
} else {
  fill(0, 0, 255);           // 青
  ellipse(200, 200, 200, 200); // 大きい
}
```

### 試してみよう

- `let x = 2;` のとき → 小さい赤い円
- `let x = 5;` のとき → 大きい青い円

変数 x の値を変えて、どう変わるか確かめてみよう！
