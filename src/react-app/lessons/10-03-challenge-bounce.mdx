## 👊チャレンジ⑩

**円が端まで行ったら、0 に戻るようにしよう**

### 目標

- 円が右下の端（x が 400 を超える）まで行ったら
- 左上（0, 0）に戻る
- 永遠に繰り返す

### ヒント

```javascript
if (x > 400) {
  x = 0;
  y = 0;
}
```

- x が 400 を超えたら、x と y を 0 にリセット
- 円が左上から再スタートする

### ポイント

- `if` 文を使って、条件をチェックする
- キャンバスのサイズは 400x400 なので、x が 400 を超えたら画面外
- リセットすることで、永遠に動き続けるアニメーションになる

## 応用

できたら、以下も試してみよう：
- 速度を変える（x = x + 2 にしてみる）
- 別の方向に動かす（x だけ増やして、横に動かす）
- 色を変える
