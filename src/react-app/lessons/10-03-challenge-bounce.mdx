## 👊チャレンジ⑩

**円が端まで行ったら、0 に戻るようにしましょう**

### 目標

- 円が右下の端（x が 400 を超える）まで行ったら
- 左上（0, 0）に戻ります
- 永遠に繰り返します

### ヒント

```javascript
if (x > 400) {
  x = 0;
  y = 0;
}
```

- x が 400 を超えたら、x と y を 0 にリセットします
- 円が左上から再スタートします

### ポイント

- `if` 文を使って、条件をチェックします
- キャンバスのサイズは 400x400 なので、x が 400 を超えたら画面外です
- リセットすることで、永遠に動き続けるアニメーションになります

## 応用

できたら、以下も試してみましょう：
- 速度を変える（x = x + 2 にしてみる）
- 別の方向に動かす（x だけ増やして、横に動かす）
- 色を変える
