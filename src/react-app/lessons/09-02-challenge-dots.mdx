## 👊チャレンジ⑨

**ネストしたループで 5x5 のドット模様を描きましょう**

### 条件

- 横に5個、縦に5個、合計 25個の円を描きます
- 等間隔で並べます
- 画面に収まるようにします

### ヒント

```javascript
for (let i = 0; i < 5; i = i + 1) {
  for (let j = 0; j < 5; j = j + 1) {
    ellipse(50 + i * 70, 50 + j * 70, 30, 30);
  }
}
```

- 外側のループで i を 0 から 4 まで変えます（横方向）
- 内側のループで j を 0 から 4 まで変えます（縦方向）
- `50 + i * 70` で横位置、`50 + j * 70` で縦位置を計算します

### 応用

できたら、以下も試してみましょう：
- 円のサイズを変える
- 間隔を変える
- 色を変える（ヒント: fill() を ellipse() の前に書く）
