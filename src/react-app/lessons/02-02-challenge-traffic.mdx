## 👊チャレンジ②

**3つの異なる色の円を縦に並べて、信号機のような絵を作ろう**

### ヒント

- 赤い円: `fill(255, 0, 0)`
- 黄色い円: `fill(255, 255, 0)`
- 緑の円: `fill(0, 255, 0)`

縦に並べるには、**x座標を同じにして、y座標だけを変える**といいよ。

例えば：
- 1つ目: `ellipse(200, 100, 80, 80)`
- 2つ目: `ellipse(200, 200, 80, 80)`
- 3つ目: `ellipse(200, 300, 80, 80)`
