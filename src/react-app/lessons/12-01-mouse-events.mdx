## マウスイベント

**マウスイベント**を使うと、クリックやマウスの動きに反応するプログラムが作れます。

### mouseClicked() 関数

```javascript
function mouseClicked() {
  // マウスがクリックされたときに実行される
  ellipse(mouseX, mouseY, 50, 50);
}
```

`mouseClicked()` は、マウスがクリックされたときに**自動的に呼ばれる**特別な関数です。

### mouseX と mouseY

```javascript
mouseX  // 現在のマウスの x 座標
mouseY  // 現在のマウスの y 座標
```

`mouseX` と `mouseY` は、p5.js が自動的に更新してくれる**特殊な変数**です。クリックした場所の座標が入っています。

### お絵描きアプリ

```javascript
function setup() {
  createCanvas(400, 400);
  background(255, 255, 255);  // setup の中に入れる
}

function mouseClicked() {
  fill(255, 0, 0);
  ellipse(mouseX, mouseY, 50, 50);
}
```

**ポイント**: `background()` を `setup` の中に入れます。
- draw の中だと、毎回背景が塗り直されて、前の円が消えます
- setup の中だと、最初に1回だけ背景を描いて、あとは円が残ります

### 他のマウスイベント

```javascript
function mouseMoved() {
  // マウスが動いたとき
}

function mouseDragged() {
  // マウスをドラッグしたとき
}

function mousePressed() {
  // マウスボタンが押されたとき
}
```

今回は `mouseClicked()` を使いますが、他にもいろいろなイベントがあります。
