## 四則演算～たす・ひく・かける・わる・あまり～

数学の四則演算や、剰余を計算できます。

### 基本的な演算子

```javascript
let a = 10;
let b = 3;

a + b; // 足し算 → 13
a - b; // 引き算 → 7
a * b; // 掛け算 → 30
a / b; // 割り算 → 3.333...
a % b; // 余り → 1 (10を3で割った余り)
```

### 計算結果を変数に代入

```javascript
let x = 2 + 3; // x は 5
let y = x * 4; // y は 20
let z = (2 + 3) * 4; // z は 20
```

括弧 `()` を使うと、計算の優先順位を変えられます。

### 剰余（余り）

`%` は**余り**を求める演算子です。割り算の余りが何か知りたいときに使います。

```javascript
10 % 3; // → 1 (10を3で割ると、3×3=9 で余りは1)
15 % 4; // → 3 (15を4で割ると、4×3=12 で余りは3)
8 % 2; // → 0 (8を2で割ると割り切れるので余りは0)
```

余りを使うと、**偶数かどうか判定**できます。偶数は 2 で割り切れる（余りが 0）からです。
