## 👊チャレンジ⑫

**クリックした場所に円を描くお絵描きアプリを作りましょう**

### 条件

- クリックした場所に円を描きます
- 何度もクリックできます
- 円が残ります（消えません）

### ヒント

```javascript
function setup() {
  createCanvas(400, 400);
  background(255, 255, 255);  // setup の中に入れる
}

function mouseClicked() {
  fill(255, 0, 0);
  ellipse(mouseX, mouseY, 50, 50);
}
```

### ポイント

- `background()` は **setup の中**に書きます
  - draw の中だと、毎回背景が塗り直されて円が消えます
- `mouseClicked()` は draw の外に書きます
  - これは p5.js が自動的に呼んでくれる特別な関数です

### 応用

できたら、以下も試してみましょう：
- 色を変える（ランダムな色にしてみる）
- 大きさを変える
- `mouseDragged()` を使って、ドラッグで線を描く
