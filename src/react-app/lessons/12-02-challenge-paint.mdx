## 👊チャレンジ⑫

**クリックした場所に円を描くお絵描きアプリを作ろう**

### 条件

- クリックした場所に円を描く
- 何度もクリックできる
- 円が残る（消えない）

### ヒント

```javascript
function setup() {
  createCanvas(400, 400);
  background(255, 255, 255);  // setup の中に入れる
}

function mouseClicked() {
  fill(255, 0, 0);
  ellipse(mouseX, mouseY, 50, 50);
}
```

### ポイント

- `background()` は **setup の中**に書く
  - draw の中だと、毎回背景が塗り直されて円が消える
- `mouseClicked()` は draw の外に書く
  - これは p5.js が自動的に呼んでくれる特別な関数

### 応用

できたら、以下も試してみよう：
- 色を変える（ランダムな色にしてみる）
- 大きさを変える
- `mouseDragged()` を使って、ドラッグで線を描く
