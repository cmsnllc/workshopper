## 繰り返してみよう

同じ処理を何度も繰り返したいとき、**ループ**を使う。

### while 文

**条件が true の間、繰り返す**。

```javascript
let i = 0;
while (i < 5) {
  print(i);
  i = i + 1;
}
// 0, 1, 2, 3, 4 と表示される
```

- `i < 5` が true の間、波括弧 `{}` の中が実行される
- `i = i + 1` で変数 i を増やす
- i が 5 になると、条件が false になって終了

### for 文

**初期化、条件、更新をまとめて書ける**。while と同じことができるけど、もっとコンパクト。

```javascript
for (let i = 0; i < 5; i = i + 1) {
  print(i);
}
// 0, 1, 2, 3, 4 と表示される
```

- `let i = 0`: 最初に1回だけ実行される（初期化）
- `i < 5`: 繰り返す条件
- `i = i + 1`: 1回のループが終わるたびに実行される（更新）

### 図形を並べる

ループを使うと、図形を等間隔で並べられる。

```javascript
for (let i = 0; i < 5; i = i + 1) {
  ellipse(50 + i * 80, 200, 50, 50);
}
```

- i が 0 のとき: `ellipse(50, 200, 50, 50)`
- i が 1 のとき: `ellipse(130, 200, 50, 50)`
- i が 2 のとき: `ellipse(210, 200, 50, 50)`
- ...

`i * 80` で、円が 80 ピクセルずつ離れて描かれる。

### 無限ループに注意

条件を間違えると、永遠に終わらない**無限ループ**になる。

```javascript
// ❌ ダメな例
let i = 0;
while (i < 5) {
  print(i);
  // i を増やすのを忘れた！
}
// i はずっと 0 のまま、永遠に終わらない
```

必ず、ループの中で変数を更新して、いつか条件が false になるようにしよう。
