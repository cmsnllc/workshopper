## 繰り返してみよう

同じ処理を何度も繰り返したいとき、**ループ**を使います。

### while 文

**条件が true の間、繰り返します**。

```javascript
let i = 0;
while (i < 5) {
  print(i);
  i = i + 1;
}
// 0, 1, 2, 3, 4 と表示される
```

- `i < 5` が true の間、波括弧 `{}` の中が実行されます
- `i = i + 1` で変数 i を増やします
- i が 5 になると、条件が false になって終了します

### for 文

**初期化、条件、更新をまとめて書けます**。while と同じことができますが、もっとコンパクトです。

```javascript
for (let i = 0; i < 5; i = i + 1) {
  print(i);
}
// 0, 1, 2, 3, 4 と表示される
```

- `let i = 0`: 最初に1回だけ実行されます（初期化）
- `i < 5`: 繰り返す条件
- `i = i + 1`: 1回のループが終わるたびに実行されます（更新）

### 図形を並べる

ループを使うと、図形を等間隔で並べられます。

```javascript
for (let i = 0; i < 5; i = i + 1) {
  ellipse(50 + i * 80, 200, 50, 50);
}
```

- i が 0 のとき: `ellipse(50, 200, 50, 50)`
- i が 1 のとき: `ellipse(130, 200, 50, 50)`
- i が 2 のとき: `ellipse(210, 200, 50, 50)`
- ...

`i * 80` で、円が 80 ピクセルずつ離れて描かれます。

### 無限ループに注意

条件を間違えると、永遠に終わらない**無限ループ**になります。

```javascript
// ❌ ダメな例
let i = 0;
while (i < 5) {
  print(i);
  // i を増やすのを忘れた！
}
// i はずっと 0 のまま、永遠に終わらない
```

必ず、ループの中で変数を更新して、いつか条件が false になるようにしましょう。
