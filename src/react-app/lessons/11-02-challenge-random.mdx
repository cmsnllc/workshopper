## 👊チャレンジ⑪

**ランダムな位置・色・大きさで円を描きましょう**

### 条件

- 位置: ランダム
- 色: ランダム
- 大きさ: ランダム（20 から 100 の範囲）

### ヒント

```javascript
let x = random(400);
let y = random(400);
let size = random(20, 100);  // 20 から 100 の範囲

fill(random(255), random(255), random(255));
ellipse(x, y, size, size);
```

- `random(20, 100)` で、20 から 100 の範囲のランダムな値
- 大きさをランダムにすると、大小様々な円が描かれます

### 応用

できたら、以下も試してみましょう：
- `background()` を削除して、円が残るようにする
  - setup の中に `background(255, 255, 255);` を移動
- frameRate を変えて、速度を調整
- 複数の円を描く（for ループを使う）
