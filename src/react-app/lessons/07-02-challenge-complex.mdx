## 👊チャレンジ⑦

**変数 x が偶数かつ5の倍数かどうか判定しよう**

### 条件

- **偶数**: 2 で割り切れる → `x % 2 == 0`
- **5の倍数**: 5 で割り切れる → `x % 5 == 0`
- **両方を満たす**: `&&` を使う

### ヒント

```javascript
x % 2 == 0 && x % 5 == 0
```

これで、x が偶数**かつ**5の倍数のときだけ `true` になる。

### 試してみよう

いろいろな値で試してみよう：

- `x = 10` → **true** （10 は偶数で、5の倍数）
- `x = 15` → **false** （15 は5の倍数だけど奇数）
- `x = 8` → **false** （8 は偶数だけど5の倍数ではない）
- `x = 20` → **true** （20 は偶数で、5の倍数）
- `x = 7` → **false** （7 はどちらも満たさない）

変数 x の値を変えて、どのときに `true` になるか確かめてみよう！
