## 比較演算子と真偽値～○○よりも大きい？小さい？～

コンピュータは2進数といって0と1の羅列で制御されています。 0 を false 、1 を true と表現します。

### 比較演算子

```javascript
let x = 10;

x > 5;   // → true  (x は 5 より大きい)
x < 5;   // → false (x は 5 より小さくない)
x >= 10; // → true  (x は 10 以上)
x <= 5;  // → false (x は 5 以下ではない)
x == 10; // → true  (x は 10 と等しい)
x != 10; // → false (x は 10 と等しくないわけではない)
```

- `>` : より大きい
- `<` : より小さい
- `>=` : 以上
- `<=` : 以下
- `==` : 等しい（**注意**: `=` は代入、`==` は比較）
- `!=` : 等しくない

### true と false

比較の結果は `true`（真）か `false`（偽）になります。これを**真偽値（ブール値、boolean）**といいます。

```javascript
println(10 > 5);  // true と表示される
println(10 < 5);  // false と表示される
```
