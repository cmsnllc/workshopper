## 比較演算子と真偽値

プログラムでは、値を比較して**真（true）**か**偽（false）**かを判定できる。

### 比較演算子

```javascript
let x = 10;

x > 5;   // → true  (x は 5 より大きい)
x < 5;   // → false (x は 5 より小さくない)
x >= 10; // → true  (x は 10 以上)
x <= 5;  // → false (x は 5 以下ではない)
x == 10; // → true  (x は 10 と等しい)
x != 10; // → false (x は 10 と等しくないわけではない)
```

- `>` : より大きい
- `<` : より小さい
- `>=` : 以上
- `<=` : 以下
- `==` : 等しい（**注意**: `=` は代入、`==` は比較）
- `!=` : 等しくない

### true と false

比較の結果は `true`（真）か `false`（偽）になる。これを**真偽値（ブール値）**という。

```javascript
print(10 > 5);  // true と表示される
print(10 < 5);  // false と表示される
```

### 偶数判定

**偶数**は 2 で割り切れる数。つまり、2 で割った余りが 0 になる。

```javascript
let x = 10;
x % 2 == 0;  // → true (10 は偶数)

let y = 7;
y % 2 == 0;  // → false (7 は奇数)
```

余り演算子 `%` と比較演算子 `==` を組み合わせると、偶数かどうか判定できるよ。
