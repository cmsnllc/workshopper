## 👊チャレンジ③

**変数 x と y の値を変えて、円を右下に移動させよう**

### ヒント

- キャンバスの座標系は、左上が (0, 0)、右下が (400, 400)
- 右に移動するには、**x の値を大きく**する
- 下に移動するには、**y の値を大きく**する

例えば：
- 中央: `x = 200, y = 200`
- 右下: `x = 300, y = 300`
- 右上: `x = 300, y = 100`
- 左下: `x = 100, y = 300`

いろいろな値を試してみて、円がどう動くか確かめてみよう！
