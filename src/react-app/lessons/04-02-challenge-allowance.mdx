## 👊チャレンジ④

**太郎と花子のお小遣い帳を完成させよう**

### ストーリー

1. 太郎と花子はそれぞれ 1000円 のお小遣いを持っている
2. **花子は太郎に500円借りる**
3. **花子は小遣いを1.2倍にする**（投資で増やした！）
4. **花子は太郎に倍の1000円返す**

最終的に太郎と花子のお小遣いはいくらになるかな？

### ヒント

- 借りる = 相手から引いて、自分に足す
  - `hanako = hanako - 500`
  - `taro = taro + 500`
- 1.2倍にする = 掛け算
  - `hanako = hanako * 1.2`
- 返す = 自分から引いて、相手に足す
  - `hanako = hanako - 1000`
  - `taro = taro + 1000`

順番に処理していけば、最終的な金額が計算できるよ！
