function setup() {
  createCanvas(l, l);
}

function draw() {
  background(255, 255, 255);
  fill(255, 0, 0);
  ellipse(x, 200, 50, 50);

  x = x + 1;

  if (x > l) {
    x = 0;
  }
}`

## 図形を動かしてみよう

ここまでは静止した図形を描画してきました。
setup は1度だけ実行されるものでしたが、draw は繰り返し実行されるという仕組みになっています。なので、draw の中を変数や制御構文を使って時間的に変化させることで図形を動かすことができます。
