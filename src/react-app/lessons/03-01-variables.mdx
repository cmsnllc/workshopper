## 変数とは

**変数**は、値を入れておく「箱」のようなもの。一度値を入れれば、何度でも使い回せる。

### 変数の宣言と代入

```javascript
let x = 200; // 変数 x に 200 を格納
let y = 200; // 変数 y に 200 を格納
```

- `let` で変数を宣言する
- `=` で値を代入する
- 変数名は自由に決められる（x, y, size, colorR など）

### 変数を使って円を描く

```javascript
let x = 200;
let y = 200;
ellipse(x, y, 100, 100); // x と y を使って円を描く
```

数値を直接書く代わりに、変数を使うことができる。変数を使うと、**あとで値を変えるのが簡単**になる。

### print() でコンソールに出力

```javascript
let x = 200;
print(x); // 左下のコンソールに "200" と表示される
print("x の値は", x); // "x の値は 200" と表示される
```

`print()` を使うと、変数の中身を確認できる。プログラムがうまく動かないときに便利だよ。
