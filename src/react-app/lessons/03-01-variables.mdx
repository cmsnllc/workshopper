## 変数～数字の入った箱～

**変数**は、値を格納しておく箱のようなものです。`let` 構文で変数を宣言できます。変数には `=` で値を代入できます。

### 変数の宣言と代入

```javascript
let x = 200; // 変数 x に 200 を格納
let y = 200; // 変数 y に 200 を格納
```

- `let` で変数を宣言します
- `=` で値を代入します
- 変数名は自由に決められます（x, y, size, colorR など）

### 変数を使って円を描く

```javascript
let x = 200;
let y = 200;
ellipse(x, y, 100, 100); // x と y を使って円を描く
```

数値を直接書く代わりに、変数を使うことができます。変数を使うと、**あとで値を変えるのが簡単**になります。

### 色を表す変数

変数は色の値を入れておくこともできます。

```javascript
let R = 255; // 赤(Red)
let G = 255; // 緑(Green)
let B = 255; // 青(Blue)
background(R, G, B);
```

上記のように書いた場合、R という変数と G という変数、B という変数を定義して、それぞれに 255 を格納しています。変数を定義すると、以降の行ではその変数を参照できるようになり、background に R と G と B の値を渡すことができるようになります。

### 変数に変数を代入する

変数に変数を代入することもできます。

```javascript
let x = 0;
let y = 10;
x = y;
println(x); // x → 10
println(y); // y → 10
```

数学では `x = y` と書くと x と y は同じ値だという意味になりますが、プログラミングでは代入する（y の値を x にコピーする）という意味になるので、注意しましょう。

### println() でコンソールに出力

変数に何が代入されているか知りたい時は `println()` を使うと下のコンソールに表示されるので、わからなくなったらコンソールに表示して確認してみるとよいでしょう。

```javascript
let x = 5;
println(x); // 5 が表示される
```

`println()` を使うと、変数の中身を確認できます。プログラムがうまく動かないときに便利です。変数の値が何か確認する時に使いましょう。
