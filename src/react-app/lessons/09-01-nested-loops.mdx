## ネストしたループ

ループの中にループを入れると、**2次元のパターン**を描ける。

### ネスト（入れ子）とは

```javascript
for (let i = 0; i < 3; i = i + 1) {
  for (let j = 0; j < 3; j = j + 1) {
    ellipse(100 + i * 100, 100 + j * 100, 30, 30);
  }
}
```

- 外側のループ（i）が1回進むごとに
- 内側のループ（j）が全部実行される

### 実行順序

i = 0 のとき、j が 0, 1, 2 と変わる：
- `ellipse(100, 100, 30, 30)` ← i=0, j=0
- `ellipse(100, 200, 30, 30)` ← i=0, j=1
- `ellipse(100, 300, 30, 30)` ← i=0, j=2

i = 1 のとき、j が 0, 1, 2 と変わる：
- `ellipse(200, 100, 30, 30)` ← i=1, j=0
- `ellipse(200, 200, 30, 30)` ← i=1, j=1
- `ellipse(200, 300, 30, 30)` ← i=1, j=2

i = 2 のとき、j が 0, 1, 2 と変わる：
- `ellipse(300, 100, 30, 30)` ← i=2, j=0
- `ellipse(300, 200, 30, 30)` ← i=2, j=1
- `ellipse(300, 300, 30, 30)` ← i=2, j=2

合計 3 × 3 = 9個の円が描かれる。

### 横と縦

- **i**: 横方向の位置（x 座標）
- **j**: 縦方向の位置（y 座標）

```javascript
ellipse(100 + i * 100, 100 + j * 100, 30, 30);
//      ↑            ↑
//      横           縦
```
