## ループで円を並べよう

for ループを使って、複数の円を等間隔で並べてみよう。

### 目標

3つの赤い円を横一列に並べる。

### ヒント

```javascript
for (let i = 0; i < 3; i = i + 1) {
  ellipse(100 + i * 100, 200, 80, 80);
}
```

- `i < 3` で3回繰り返す（i = 0, 1, 2）
- `100 + i * 100` で x 座標を計算
  - i = 0 のとき: 100 + 0 * 100 = 100
  - i = 1 のとき: 100 + 1 * 100 = 200
  - i = 2 のとき: 100 + 2 * 100 = 300
