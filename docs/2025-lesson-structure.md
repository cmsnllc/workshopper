# 2025年度 レッスン構成案

## 基本方針

- **説明パート**: 理論的な内容、概念の説明（コードブロックあり、実行不要）
- **実践パート**: コードを書いて試す（エディタ + プレビュー）
- 簡単なレッスンは1つのMDXファイルで完結、複雑なレッスンは複数に分割

## 構成パターン

### パターンA: 説明 + 実践（1ファイル）
説明が簡単で、すぐに実践できるレッスン

### パターンB: 説明パート → 実践パート（複数ファイル）
説明が長く、概念理解が重要なレッスン
